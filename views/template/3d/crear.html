Animación 3D

Crear animación utilizando por modelo de esqueletos

Sofware a utilizar:

Blender

1.- Iniciamos digitalizando un bosquejo dibujado a mano como referencia.

2.- Modelado del personaje.

3.- Se agregan colores y texturas al modelo.

4.- Se crean las uniones que tendrá el esqueleto para poder animarlo.

5.- El modelo tiene capacidad de movimiento en sus extremidades.

6.- Agregar parámetros a las extremidades con una línea de tiempo.

7.- Así podemos hacer que el modelo ejemplifique la animación de correr.
